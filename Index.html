Paso 1: El Punto de Paso (P)

El punto me lo dan servido
ACCIÓN: Ya tenés P = (x0, y0, z0).
P es la INTERSECCIÓN entre Recta y Plano
ACCIÓN: 1) Parametrizá la recta. 2) Sustituí x, y, z en la ec. del plano. 3) Hallá Lambda. 4) Volvé a la recta para sacar el Punto.

Paso 2: El Vector Normal (n)

El plano es PERPENDICULAR a una recta "r"
ACCIÓN: ¡Golazo! El vector director de la recta ES la normal del plano. n = v_r.
El plano es PARALELO a otros dos vectores (o contiene a dos rectas)
ACCIÓN: Hacé Producto Vectorial entre los dos vectores para fabricar la Normal.

Paso 3: Ensamblado (Ecuación General)

Tengo mi Punto P y mi Normal n = (A, B, C)
ACCIÓN: Escribí A(x - x0) + B(y - y0) + C(z - z0) = 0. Luego distributiva y listo.

Rama 1: Calculo de DISTANCIAS

Distancia de un PUNTO a un PLANO
ACCION: Usar formula directa.
d = |A(x0) + B(y0) + C(z0) + D| / √(A² + B² + C²)
Distancia entre DOS PLANOS
ACCION PREVIA: Verificar que sean paralelos (si no lo son, la distancia es 0).
ACCION: Igualar los coeficientes A, B, C en ambos planos.
d = |D1 - D2| / √(A² + B² + C²)
Distancia de un PUNTO a una RECTA
ACCION: Armar vector desde un punto de la recta al punto externo (AP).
d = || Vector_Director x AP || / || Vector_Director ||
Distancia entre DOS RECTAS (Alabeadas)
ACCION: Armar vector entre los dos puntos de las rectas (P1P2).
Hacer producto mixto: d = | P1P2 · (v1 x v2) | / || v1 x v2 ||

Rama 2: Calculo de ANGULOS

Angulo entre DOS RECTAS
ACCION: Extraer los vectores directores (v1 y v2).
cos(α) = |v1 · v2| / ( ||v1|| * ||v2|| )
Angulo entre DOS PLANOS
ACCION: Extraer los vectores normales (n1 y n2).
cos(α) = |n1 · n2| / ( ||n1|| * ||n2|| )
Angulo entre una RECTA y un PLANO
ACCION CRITICA: Aca se usa SENO, no coseno.
Extraer vector director (v) y normal (n).
sen(α) = |v · n| / ( ||v|| * ||n|| )
